Organize Net Device Help

Use this page to set the interconnections between devices on an account.

Reading the diagram

This page shows all of the devices in the current account.

Devices on the top row of the diagram are not behind any other devices in the account. An arrow from device A to device B indicates that device B is behind device A. Multiple arrows pointing to a single device indicate that the device is behind multiple devices.

Devices that are behind the same set of "parent" devices are grouped together within a box-outline. This box will have one set of arrows pointing to the middle device in the group, but the relationship that set of arrows signifies applies to all of the devices within that box.

To see more information about a particular device, hover over that device in the diagram.

Organizing the current device

Use the tabular form on the left-hand side of the page to select which devices are behind this device in the network. You can also select and unselect devices in this table by clicking on their image in the diagram. Note only devices with a status of at least Contract Received will appear in this table.

To save your changes to the organization of a device, click the set button. The page will redisplay with the information on the page updated to reflect your changes.
